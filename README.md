# 《乐动健康》—— HTML5与移动互联网应用开发实训二

## 项目介绍
“乐动”是一款专注于身体健康管理的微信小程序，设有跑步、瑜伽等多种运动项目，支持记录用户的运动和饮食数据，帮助用户养成健康生活习惯。
使用即时原型工具进行产品原型图设计，参与功能流程与界面交互规划。
使用 ECharts 实现运动数据的图表可视化，提升用户对健康数据的感知。
后台管理系统采用Vue.js框架及Element UI组件库开发，实现数据管理功能。
